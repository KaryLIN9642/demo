----- HTML5 語意標籤（Semantic Elements）-----

  • 用意是：取代掉不具意義的 < div >等tag
  • 用語意化標籤來加強文件的結構化

header：網頁的標頭，通常放置網站標題。

nav：網頁的選單、導覽，之前寫法為 < div class=“nav” >。

main：網頁的主要內容。一個html只能有一個main標籤，網頁主要的內容可用main標籤標記。

article：一篇文章內容。

section：自訂的區塊，例如數篇摘要組成的空間。

aside：除了內容之外的其他內容。例如：側邊欄。aside 的內容應該與周圍的內容有相關性。

figure：是一個有完整內容的區塊，是主要內容的一部分，可以任意移動位置而不影響整體內容的表達。例如：圖片與標題可以組合在 figure 元素裡

footer：網頁的頁尾，通常放置聯絡方式、著作權宣告等等。

mark：強調一小塊內容。

time：顯示日期時間。

二、CSS

主要功能在於裝飾HTML的顯示內容，可於head元素內插入style標籤，或將CSS設定獨立成.CSS的格式檔案，只要在原本HTML的head元素內鍵入連結路徑即可。

TIP：基本格式、Box Model、CSS選擇器、定位模式、網頁排版，其他標籤使用介紹可詳w3schools.com

----- 基本格式 -----

div區塊標籤
span行內標籤 br/強迫斷行

----- Box Model -----

margin表示2 div各區塊間的距離：本區塊與上下左右都有30px距離
設定如下：style="margin:30px"
本區塊與上面區塊40px距離，因為上面區塊已經設定30px， 因此重疊的部份會以較大為主，因此倆區塊之間距離40px。
設定如下：style="margin-top:40px"

border的呈現，這是正常的狀況
設定如下：style="border:1px dashed #336666"
border邊框變成圓角，邊距內文字與邊距之距離
設定如下： style="border:1px dashed #613030; border-radius:12px; padding:10px"

width寬
設定如下：style="padding:10px; width:400px"
height高
設定如下：style="padding:10px; height:50px"

綜合上述說明，最後呈現結果！

----- CSS選擇器 -----

上述針對css的設定都寫寫在div標籤內，排版上會顯得雜亂，重點是若有相同標籤，性質無法統一修改， 因此可把所有css的設定，集中管理，僅需在head標籤下，開一個style標籤，把所有css設定都放在style標籤內。

  • 標籤名稱選擇器

    • 假設針對body標籤做設定，則css設定如下：
    <style type="text/css">
    body{font-family:"微軟正黑體"}
    </style>

    此設定會統一body標籤內所有內容的字形


  • class選擇器

    • 假設div標籤內class的名稱為name，則css設定如下：
    <style type="text/css">
    .name{ margin:30px; border:1px dashed #336666; padding:10px; }
    </style>

    需在div標籤設定一個特定的名稱，如此在css的設定才會套用到所指定的內容。設定如下：
    <div class="name"></div>

    若只須設定某個div標籤下class="name"名稱，css設定如下：
    div.name{ margin:30px; border:1px dashed #336666; padding:10px; }


  • id選擇器

    • 假設對下面"按鈕"做設定，按鈕名稱設定成id="btn"，則css設定如下：
    #btn{border:1px solid #888888;background-color:#ffffff;padding:5px;}

    填寫訊息

  • 虛擬選擇器：搭配其他選擇器使用

    • 假設滑鼠移到上面"按鈕"時，按鈕顏色變成灰色，則css設定如下：
    #btn:hover{background-color:#888888;}

    當滑鼠點入input輸入框時，邊框顏色變成紅色，則css設定如下：
    :input:focus{border:2px solid #ff0000;}


  • 階層選擇器：搭配其他選擇器使用

    • 要變更某個class=pagetop標籤下的超聯結時，則css設定如下：
    .pagetop a{color: #ffffff;}

----- 定位模式 -----

預設的位置，按照程式碼的順序排列

絕對定位：原本的Box會以網頁上端與左端為基準，跑到指定的位置，這是完全獨立的，不影響版面，但會遮住下面的文字
設定為：position:absolute; left:200px;

絕對定位
相對定位：原本的Box會以目前位置危機準，跑到相對的位置，原本位置會空出來，會影響版面排版。因此常用於小區塊的位移
設定為：position:relative; left:200px;
