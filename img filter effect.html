1. 轉換為灰度圖像：grayscale( % ), 0％ (0)是預設的，表示原始圖像。100％將會使圖像完全灰化(used for black and white images)。例子：grayscale 0%, 50%, 100%
2. 應用模糊效果的圖像：blur( px ), 較大的值會產生更多的模糊, 例子：blur 1px, 2px, ,3px, 4px, 5px
backdrop-filter: blur 5px
3. 調整圖像的亮度：brightness( % ), 0％將使圖像全黑, 100％ (1)是預設的，表示原始圖像。超過100％的值將提供更亮的結果。 例子：brightness 50%, 150%, 200%, 300%
backdrop-filter:brightness 50%
4. 調整圖像的對比度：contrast( % ), 0％將使圖像全黑, 100％ (1)是預設的，表示原始圖像。超過100％的值將提供的結果與較少的對比。 例子：contrast 50%, 150%, 200%, 300%

5. 適用陰影效果的圖像：drop-shadow( h-shadow v-shadow blur spread color ),
H-陰影 -必需。指定水平陰影的像素值。 負值陰影放置在圖像的左側。
V-陰影 -必需。指定垂直陰影的像素值。 負值放置圖像上方的陰影。
blur-可選。必須以像素為單位。添加一個模糊效果的陰影。較大的值會產生更多的模糊（陰影變得更大，更輕）。注意：負值是不允許的。如果沒有指定值，默認值為0（陰影的邊緣銳利）。
spread-可選。必須在像素。正值將使陰影擴展和增長更大，負值將使陰影收縮。如果未指定，這將是0 (the shadow will be the same size as the element)。
color-可選。如果未指定，則顏色取決於瀏覽器(often black)。 創造一個紅影，其水平和垂直方向為8像素大，與10px的的模糊效果的例子：drop-shadow(8px 8px 10px red),(0px 0px 10px red), (10px 0px 15px red)
backdrop-filter:drop-shadow (20px 10px red)
6. 應用該圖像上的色調旋轉：hue-rotate( deg ),應用於所述圖像上的色調旋轉。該值定義沿色環圖像樣本將被調整的度數。0deg是預設值，表示原始圖像。最大值為360deg 例子：hue-rotate 90deg, 180deg, 270deg, 360deg
7. 反轉圖像中的樣本：invert( % ), 0％ (0)是預設的，表示原始圖像。100％將使圖像完全反轉。注意：負值是不允許的。 例子：invert 10%, 20%, 40%, 80%, 100%
8. 設置圖像的透明度級別：opacity( % ), 0％是完全透明的。100％ (1)是預設的，表示在原始圖像(no transparency)。注意：負值是不允許的。 例子：opacity 0.1, 0.2, 0.5, 0.7, 0.9
9. 飽和的圖像：saturate( % ), 0％ (0)將使圖像完全不飽和的。100％是預設和表示原始圖像。超過100％的值提供超飽和的結果。注意：負值是不允許的。 例子：saturate 1.5, 2, 5, 8
10. 將圖像轉換為棕褐色：sepia( % ), 0％ (0)是預設的，表示原始圖像。100％將使圖像完全棕褐色。注意：負值是不允許的。 例子：sepia 0.1, 0.2, 0.5, 0.8, 1
11. 合併過濾器：要使用多個過濾器，每個濾波器用空格分隔。注：順序很重要(ie using grayscale()後， sepia()將導致一個完全灰度圖像） 例子：contrast(200%) brightness(150%)

13. backdrop-filter效果：
backdrop-filter:blur()：用於將高斯模糊應用於圖像。此功能的預設值為0，不應用任何模糊效果。 |
brightness()：用於使圖像更亮或更暗。值超過100％將使圖像變亮，而值低於100％將使圖像變暗。如果亮度變為0％，它將完全使圖像變黑。|
contrast()：它用於設置圖像的對比度。原始圖像的對比度為100％。如果對比度低於0％，將使圖像完全變黑 |
drop-shadow()：用於對元素應用陰影效果。它接受水平和垂直陰影量以及散布和顏色值。 |
grayscale()：用於將圖像的顏色轉換為黑白。值為0％表示原始圖像，值為100％表示完全黑白圖像。 |
hue-rotate()：用於將色調旋轉應用於圖像。函數值表示將調整圖像圓的色環周圍的度數。預設值為0，表示原始圖像。 |
invert()：用於反轉圖像。預設值是0％，代表原始圖像，而100％將使圖像完全反轉。 |
opacity()：它用於設置圖像的不透明度。預設值為0％，表示圖像是完全透明的，而100％的值表示原始圖像是完全不透明的。|
saturate()：用於設置元素的飽和度。預設值為100％，表示原始圖像。 0％的值表示完全不飽和的圖像，而100％以上的值表示super-saturated圖像 |
sepia()：它用於將圖像轉換為棕褐色，使圖像看起來更溫暖。 0％的值表示原始圖像，而100％的值表示完整的棕褐色圖像。 |
none | initial:它用於將此屬性設置為其預設值。 | inherit:它從其父元素繼承屬性。
